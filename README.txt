# README.txt


This is an executable JAR file for the project "MP1 CPU Scheduling". 

## Prerequisites

- Java Runtime Environment (JRE) version 8 or higher must be installed on your system.

##

1. Download the "MP1-CPU-Scheduling-.jar" file.
2. Ensure that Java is installed on your system. You can download Java from the official website: https://www.oracle.com/java/technologies/javase-downloads.html

## Run the jar file

1. Open a Command Prompt (Windows).

2. Navigate to the directory where the JAR file is located. For example, if the file is in your Downloads folder, you would use the following command:

   **Windows:**
   
   cd C:\Users\YourName\Downloads

3. java -jar MP1-CPU-Scheduling-.jar

4. The application should start running.
